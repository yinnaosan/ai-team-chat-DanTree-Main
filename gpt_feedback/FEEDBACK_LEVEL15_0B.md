## 2. WHAT WAS BUILT

### `multiEntityComparisonEngine.ts`
Pure comparison aggregator — no new data fetching, wraps existing engines:
- `buildMultiEntityComparison(entityA, entityB)` → `MultiEntityComparisonResult`
- 5 comparison dimensions: `semantic_direction`, `evidence_strength`, `output_gate`, `source_breadth`, `fragility`
- Each dimension has `winner: "left" | "right" | "tie" | "unavailable"`, `left_value`, `right_value`
- `comparison_summary`: human-readable advisory string
- `advisory_only: true` always set

### `market.compareEntities` tRPC query
- Input: `{ entityA: string, entityB: string }` (max 20 chars each)
- Returns: `{ available: true, ...MultiEntityComparisonResult }` or graceful fallback
- Public procedure, no auth required
